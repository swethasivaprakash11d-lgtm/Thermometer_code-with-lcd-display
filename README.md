# Thermometer with LCD Display using Arduino UNO

## Overview

This project implements a digital thermometer using Arduino UNO, a temperature sensor, and a 16x2 LCD display. The measured temperature is continuously monitored and displayed on the LCD screen in real time.

## Components Used

* Arduino UNO
* Temperature Sensor (LM35 or equivalent)
* 16x2 LCD Display
* Breadboard
* Jumper Wires
* Potentiometer (for LCD contrast)

## Working Principle

The temperature sensor measures the surrounding temperature and sends the data to the Arduino. The Arduino processes the sensor readings and converts them into temperature values. The measured temperature is then displayed on the 16x2 LCD screen.
